# nis2-simulation
NIS2 Ledelsessimulation Hvad gør du, når ransomware rammer kl. 08.12 – og myndighederne venter på svar inden for 24 timer? Træf ledelsens beslutninger i realtid og se konsekvenserne live: bødeeksponering, personligt direktøransvar og juridiske forpligtelser efter NIS2-loven. 11 scenarier. Ca. 20 minutter.
